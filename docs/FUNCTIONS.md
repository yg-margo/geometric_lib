# Справочник функций

## circle.py

### area(r)
Вычисляет площадь круга по радиусу.

- **Параметры:**  
  r (float): радиус круга  
- **Возвращает:**  
  float: площадь круга  
- **Пример:**
  ```
  area(5)   # 78.53981633974483
  ```

### perimeter(r)
Вычисляет длину окружности по радиусу.

- **Параметры:**  
  r (float): радиус круга  
- **Возвращает:**  
  float: длина окружности  
- **Пример:**
  ```
  perimeter(5)   # 31.41592653589793
  ```

- **Возвращает:**  
  float: периметр  
- **Пример:**

## triangle.py

### area(a, h)
Вычисляет площадь треугольника по основанию и высоте.

- **Параметры:**  
  a (float): основание  
  h (float): высота  
- **Возвращает:**  
  float: площадь  
- **Пример:**
  ```
  area(6, 4)  # 12.0
  ```

### perimeter(a, b, c)
Вычисляет периметр треугольника по трем сторонам.

- **Параметры:**  
  a (float): первая сторона  
  b (float): вторая сторона  
  c (float): третья сторона  
- **Возвращает:**  
  float: периметр  
- **Пример:**
  ```
  perimeter(3, 4, 5)  # 12
  ```
